### Local setup instructions
- npm install 
- Run a postgres DBMS locally.
- Create a .env in root directory and configure the following variables with your postgres variables respectively. 
```
DB_HOST
DB_USER
DB_PASSWORD
DB_NAME
```
- npm run knex migrate:latest
- npm run devStart
